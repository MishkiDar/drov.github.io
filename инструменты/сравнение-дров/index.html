Сравнение для разных целей:

  • Для отопления дома - дуб (максимальная теплота), береза (оптимальное соотношение), сосна (экономичный вариант)
  • Для бани - ольха (легкий пар), береза (жаркое пламя), осина (очистка в конце)
  • Для камина - дуб (долгое горение), береза (красивое пламя), ольха (мало дыма)
  • Для копчения - ольха, осина, фруктовые породы (без смолы)
  • Для мангала - береза, дуб (жаркое горение), ольха (стабильный жар)

Детальное сравнение популярных пород:

Дуб vs Береза:

  1. Дуб дает на 5% больше тепла, но стоит на 20% дороже
  2. Береза быстрее разгорается, дуб горит дольше
  3. Дуб идеален для долгого горения, береза - для быстрого разогрева
  4. Береза доступнее и популярнее

Ольха vs Осина:

  • Ольха дает на 8% больше тепла чем осина
  • Ольха лучше для бани и копчения
  • Осина дешевле и очищает дымоход
  • Оба варианта дают мало дыма

Рекомендации по выбору:

  1. Для экономии - выбирайте сосну или смешанные дрова
  2. Для максимальной теплоотдачи - дуб или береза
  3. Для бани - комбинация ольхи и березы
  4. Для камина - дуб для долгого горения, береза для красоты
  5. Универсальный вариант - береза (лучшее соотношение)
