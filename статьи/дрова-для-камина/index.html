Дрова для камина: выбор породы и правильное использование

Идеальные дрова для камина: красивое пламя, долгое горение, приятный аромат. Выбор породы, размера и влажности для открытого и закрытого камина.

Какие дрова лучше для камина

Выбор дров для камина отличается от выбора дров для печи. Камин - это не только источник тепла, но и элемент интерьера, создающий уютную атмосферу. Поэтому важны не только теплоотдача и время горения, но и внешний вид пламени, аромат и количество искр.

Лучшие породы дров для камина:

  1. Дуб - долгое горение, жаркое пламя, минимум искр, премиум выбор
  2. Береза - красивое пламя, приятный аромат, быстро разгорается
  3. Ольха - мало дыма, легкий аромат, идеальна для открытого камина
  4. Ясень - жаркое пламя, долгое горение, мало искр
  5. Бук - равномерное горение, хорошая теплоотдача
  6. Вишня - приятный аромат, красивое пламя
  7. Яблоня - фруктовый аромат, мягкое пламя

Породы дров которые НЕ подходят для камина:

  • Хвойные породы - сосна, ель дают много искр и смолы, опасны для открытого камина
  • Сырые дрова - влажность выше 20% дает много дыма и мало тепла
  • Обработанная древесина - краска, лак выделяют токсины при горении
  • Мелкие опилки - быстро сгорают, не дают красивого пламени
  • Гнилая древесина - неприятный запах, мало тепла

Требования к дровам для камина:

Размер поленьев

  • Длина: 25-35 см
  • Толщина: 8-12 см
  • Одинаковый размер
  • Правильная форма

Влажность

  1. Идеально: 15-20%
  2. Максимум: 25%
  3. Сухие дрова горят лучше
  4. Меньше дыма

Качество

  • Без гнили
  • Без плесени
  • Без коры (опционально)
  • Правильно наколотые

Особенности дров для открытого камина:

  1. Мало искр - дуб, ольха, ясень безопаснее хвойных
  2. Мало дыма - сухие дрова лиственных пород
  3. Красивое пламя - береза, дуб дают красивое пламя
  4. Приятный аромат - фруктовые породы, ольха
  5. Долгое горение - дуб, ясень горят дольше
  6. Равномерное горение - одинаковый размер поленьев

Особенности дров для закрытого камина:

  • Высокая теплоотдача - дуб, береза дают больше тепла
  • Долгое горение - плотные породы горят дольше
  • Меньше требований к искрам - закрытый камин безопаснее
  • Эффективность важнее красоты - можно использовать хвойные
  • Регулируемая тяга - можно использовать разные породы

Техника топки камина дровами:

  1. Очистите камин от золы от предыдущей топки
  2. Проверьте тягу - откройте заслонку дымохода
  3. Уложите растопку (береста, щепа) в центре
  4. Добавьте мелкие поленья крест-накрест
  5. Положите крупные дрова сверху
  6. Разожгите растопку снизу
  7. Регулируйте подачу воздуха заслонкой

Секреты красивого пламени:

  • Используйте сухие дрова - пламя ярче
  • Правильная укладка - обеспечивает равномерное горение
  • Добавляйте дрова постепенно - поддерживайте пламя
  • Используйте породы с красивым пламенем - береза, дуб
  • Контролируйте тягу - слишком сильная тяга уносит пламя

Расход дров для камина:

  • Открытый камин - 5-8 кг дров в час, много тепла уходит в трубу
  • Закрытый камин - 3-5 кг дров в час, более эффективен
  • Камин-печь - 2-4 кг дров в час, максимальная эффективность
  • Вечерняя топка (3-4 часа) - 15-25 кг дров для открытого камина
  • Сезонный расход - зависит от частоты использования

Хранение дров для камина:

  1. Храните дрова в сухом проветриваемом месте
  2. Используйте дровницу рядом с камином для удобства
  3. Подготовьте запас на 2-3 топки заранее
  4. Проверяйте влажность перед использованием
  5. Храните разные породы отдельно для комбинирования
